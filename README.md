
# Python Tkinter Projects

A collection of GUI applications and Tkinter practice projects developed using Python. This repository documents my learning journey from basic Tkinter widgets to building complete desktop applications.

## Technologies Used

- Python
- Tkinter
- Pillow (PIL)
- SQLite3
- Object-Oriented Programming

---

## Projects Included

### Tkinter 1 – GUI Fundamentals
- Creating and displaying labels
- Grid layout management
- Button widgets
- Event handling using commands
- User input with Entry widgets

### Tkinter 2 – Simple Calculator
- Built a calculator GUI using Tkinter
- Supports:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Implemented button-based user interaction

### Tkinter 3 – Image Viewer Basics
- Displaying images in a Tkinter window
- Loading images using Pillow (PIL)
- Custom application icons
- Window management

### Tkinter 4 – Image Gallery Viewer
- Built an image slider/gallery
- Previous and Next navigation buttons
- Dynamic image switching
- State management for navigation controls

### Tkinter 5 – To-Do List Application
- Complete GUI-based task manager
- Add tasks
- Update tasks
- Delete tasks
- Mark tasks as completed
- User-friendly interface with icons and background images
- Implemented using Object-Oriented Programming

### Tkinter 6 – Advanced Widgets
- Label Frames
- Radio Buttons
- Dynamic option selection
- Event-driven programming
- Widget grouping and layout management

### Tkinter 7 – Utility Widgets
- File Explorer using File Dialog
- Image selection and display
- Scale (Slider) widgets
- Dynamic window resizing
- Checkboxes and user preferences

### Tkinter 8 – SQLite Address Book Database
- Integrated SQLite database with Tkinter
- Created and connected a local database
- Built an Address Book application
- Added records using GUI forms
- Stored contact information permanently
- Displayed saved records from the database
- Deleted records using record IDs
- Performed database insertion and retrieval operations
- Learned database connectivity using Python's sqlite3 module

### Tkinter 9 – Address Book Record Editor
- Extended the Address Book application with update functionality
- Retrieved existing records from the SQLite database
- Selected records using unique IDs
- Opened a separate editing window
- Updated existing contact information
- Applied SQL UPDATE queries through Tkinter
- Implemented complete CRUD operations:
  - Create
  - Read
  - Update
  - Delete
- Managed multiple windows and database interactions
- Built a fully functional database-driven desktop application
